# IM 即时通讯系统 - 项目路线图

## 项目概述
打造一个完整的即时通讯系统，包含：
- **im-backend**: Java Spring Boot 后端服务
- **im-desktop**: Tauri + JS 桌面客户端
- **im-mobile**: Flutter 移动端（Android/iOS）

## 阶段一：基础通讯架构
### 1.1 后端基础服务
- [ ] 用户注册与登录（JWT）
- [ ] WebSocket 长连接服务
- [ ] 基础消息收发（文本消息）
- [ ] 好友关系管理
- [ ] 单聊功能

### 1.2 桌面端基础功能
- [ ] Tauri 项目搭建
- [ ] 用户登录界面
- [ ] 联系人列表
- [ ] 聊天窗口
- [ ] 消息发送与接收

### 1.3 移动端基础功能

- [ ] 用户登录界面
- [ ] 联系人列表
- [ ] 聊天窗口
- [ ] 消息发送与接收

## 技术选型
- **后端**: Java 17+, Spring Boot 3.x, MySQL, Redis, RabbitMQ
- **桌面端**: Tauri 2.x, Vanilla JS, HTML/CSS
- **移动端**: Flutter 3.x, Dart
- **协议**: WebSocket + JSON, HTTP REST API
