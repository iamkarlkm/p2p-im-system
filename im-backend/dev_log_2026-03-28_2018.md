## 核心功能实现

### 1. 意图识别引擎
- 支持5种意图类型：SEARCH/NAVIGATE/INQUIRE/COMPARE/BOOK
- 基于关键词匹配的意图检测
- 上下文继承机制（多轮对话）
- 置信度评分

### 2. 实体提取
- 位置实体（附近/周边）
- 分类实体（火锅/烧烤/西餐等）
- 价格实体
- 时间实体（今天/明天/周末）

### 3. 多轮对话管理
- 会话上下文缓存
- 支持最多10轮对话历史
- 会话过期机制（30分钟）

### 4. 自然语言回复生成
- 基于意图的动态回复模板
- 搜索结果整合到回复
- 智能推荐原因说明

### 5. 语义POI搜索
- 自然语言查询理解
- 多维度筛选（距离/评分/价格/营业状态）
- 个性化排序预留

### 6. API端点
```
POST   /api/v1/local/assistant/dialog          # 智能对话
POST   /api/v1/local/assistant/dialog/async    # 异步对话
POST   /api/v1/local/assistant/semantic-search # 语义搜索
GET    /api/v1/local/assistant/quick-search    # 快速搜索
GET    /api/v1/local/assistant/poi/{id}/qa     # POI问答
POST   /api/v1/local/assistant/voice-search    # 语音搜索
```
