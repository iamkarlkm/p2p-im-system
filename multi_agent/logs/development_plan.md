# 开发计划列表

记录每个功能的开发状态：已完成、开发中、待开发

## 格式

```markdown
## 功能列表

| 功能名称 | 所属模块 | 状态 | 创建时间 | 完成时间 |
|----------|----------|------|----------|----------|
| 功能名1 | 后端/桌面/移动 | 已完成/开发中/待开发 | 日期 | 日期 |
| 功能名2 | 后端/桌面/移动 | 已完成/开发中/待开发 | 日期 | - |
```

## 使用说明
- 开发代理每次执行时，先读取此文件
- 从架构代理的 roadmap.md 获取新功能
- 更新功能状态
- 完成后记录完成时间
